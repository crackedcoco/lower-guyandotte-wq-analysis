Technical Details

Methodology

WV 47CSR2 Standards

Analysis applies West Virginia-specific water quality criteria rather than generic EPA standards. Key differences include Fe: 1.5 mg/L, pH: 6.0-9.0, and Al Dissolved: 0.75 mg/L.

Ambient vs Point Source

Data filtered to exclude point source discharge monitoring. Ambient stream samples identified by AU_ID patterns not containing "SOURCE" designation.

Non-Detect Handling

Left-censored data processed using Kaplan-Meier survival analysis for summary statistics. Half-detection-limit substitution used for trend analysis.

Fecal Coliform Protocol

Two-part standard assessment: 200 CFU/100mL geometric mean calculated per station, plus 400 CFU/100mL single-sample maximum threshold.

Trend Analysis

Mann-Kendall test for monotonic trends with Sen's slope estimator for magnitude. Significance threshold p < 0.05.

Hotspot Scoring

Composite impairment score per HUC12 based on parameter-specific exceedance counts weighted by ecological severity.
